# **App Name**: AdWiseAI

## Core Features:

- KPI Estimation: Estimate reach, CTR, and confidence score based on input data and campaign parameters. Determine appropriate budget level.
- Campaign Overview Generation: Generate a concise campaign title and summary outlining the advertising strategy and primary objective using AI tool.
- Geo-Strategy Recommendation: Recommend optimal radius and target regions based on business location and customer profiles, suggest ideal advertising times.
- Audience Segment Creation: Define distinct audience segments, detailing demographics, interests, and preferred channels without sensitive traits.
- Creative Ad Generation: Create ad variations tailored to specific audience segments, incorporating geo-contextual hooks and relevant calls to action. Ad copy generation assisted by AI tool.
- Data Visualization: Visualize KPI estimates, geographical strategies, audience segments, and creative ads using charts, maps, and interactive elements within a web dashboard.
- Dashboard UI: Enable easy navigation and interaction, making ad campaign planning simple. Offer options to configure user input.

## Style Guidelines:

- Primary color: Deep blue (#2962FF), suggesting strategy and focus.
- Background color: Light blue-gray (#E8F0FE) to maintain a clean, professional feel.
- Accent color: Orange (#FF9100) for important CTAs and visual alerts.
- Body and headline font: 'Inter', sans-serif, with clean, machined letterforms for an objective and neutral feel.
- Use minimalist icons for KPIs and targeting options, indicating specific actions.
- Cards layout and sidebars for different settings and inputs.
- Smooth transitions for dashboard elements, offering feedback without distractions.
